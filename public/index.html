Job Board API

Server is running! Use the API endpoints below.

Authentication

  • POST /api/auth/register
  • POST /api/auth/login

Employers

  • GET /api/employers
  • POST /api/employers (admin only)
  • GET /api/employers/:id
  • PATCH /api/employers/:id (admin only)
  • DELETE /api/employers/:id (admin only)

Jobs

  • GET /api/jobs
  • POST /api/jobs (employer only)
  • GET /api/jobs/:id
  • PATCH /api/jobs/:id (owner/admin)
  • DELETE /api/jobs/:id (owner/admin)

Candidates

  • POST /api/candidates
  • GET /api/candidates/:id (employer/admin)
  • GET /api/candidates (employer/admin)

Applications

  • POST /api/applications
  • GET /api/applications/job/:jobId (employer)
  • PATCH /api/applications/:id/status (employer)

Notifications

  • GET /api/notifications
  • PATCH /api/notifications/:id/read
  • POST /api/notifications (admin only)

Admin

  • GET /api/admin/counts
  • GET /api/admin/applications/status
  • GET /api/admin/applications/per-job
  • GET /api/admin/applications/monthly
